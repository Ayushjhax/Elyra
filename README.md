# Horizon Landing Page

A beautiful, modern landing page for Horizon - an AI-powered forecasting and hedging platform.

## Features

- Full-screen video background
- Responsive design for all devices
- Interactive FAQ section
- Beautiful typography using Space Grotesk and Inter fonts
- Smooth animations and transitions

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/` - Next.js app directory with pages and layouts
- `public/` - Static assets (logo.png, video.mp4)
- `app/page.tsx` - Main landing page
- `app/layout.tsx` - Root layout with fonts
- `app/globals.css` - Global styles and Tailwind configuration

## Build for Production

```bash
npm run build
npm start
```
